# Kurd Designer

Kurd Designer وێبسایتێکی سادە و جوانە بۆ وێنەکاری و دیزاین، بە زمانی کوردی، بە شێوازێکی هاوشێوەی Photoshop.

## تایبەتمەندییەکان
- پەڕە، سڕەر، هێڵ، چوارگۆشە، بازنە، نووسین
- بارکردنی وێنە
- فلتەرە خێراکان
- Undo / Redo
- Zoom
- هەناردەکردنی کار وەک PNG و JPG
- پاشەکەوت و کردنەوەی پرۆژە وەک JSON

## ڕێگای کارکردن لەسەر لوکال
فایلەکان بکەوە و `index.html` بکەرەوە لە وێبگەڕەکەت.

## بڵاوکردنەوە لە Vercel
1. فایلەکان بخەرە GitHub.
2. لە Vercel پرۆژەیەکی نوێ دروست بکە.
3. GitHub Repo ـکەت هەڵبژێرە.
4. Framework preset: `Other`
5. Deploy بکە.

## تێبینی
ئەم وەشانە Photoshop ـی تەواو نییە، بەڵام زۆربەی ئامرازە سەرەکییەکانی وێنەکاری و دیزاینی تێدایە و بۆ دەستپێک و بڵاوکردنەوە زۆر گونجاوە.
